Axios

Axios​

Axios是一个基于promise的网络请求库;

js引入地址：​

使用方式：​

直接调用axios发送请求;

axios(config配置对象);

axios({

    method:"请求方式",
    
    url:"请求地址",

    data:{}

})

.then((res)=>{

    //res是axios封装过的

    //axios默认只会在响应状态为2xx时才会调用then

    console.log(res);

})

.catch((err)=>{

    console.log(err);

})

axios的配置对象​

1.baseURL 指定服务器的根目录（路径的前缀）;

2.url 请求地址（必要配置）;

3.method 请求方式，默认是get;

4.headers:{} 指定请求头;

5.data:{} 请求体;

6.transformRequest[] 可以用来处理请求数据（data）;

它需要一个数组作为参数，数组可以接收多个函数;

请求发送时多个函数会按照顺序执行;

函数在执行时，会接收到两个参数data和headers;

最后一个函数必须返回一个字符串，才能使得数据有效;

7.params 用来指定路径中的查询字符串;

8.timeout 过期时间;

9.signal 用来终止请求;

axios的默认配置​

配置默认地址：

axios.defaults.baseURL ="http://...";

配置默认请求头：

axios.defaults.headers.common['Authorization'] = 
AUTO_TOKEN;

axios的实例​

1.axios实例相当于是axios的一个副本它的功能和axios一样;

const instance = axios.create();

instance.get("/...").then().catch();

2.axios的默认配置在实例也同样会生效;

但是我们可以单独修改axios实例的默认配置;

const instance = axios.create({

    baseURL:"http://...";

});

const instance = axios.create();

instance.defaults.baseURL = "http://...";

axios的拦截器​

1.axios的拦截器可以对请求或者响应进行拦截;

在请求发送前和响应读取前处理数据;

2.添加请求拦截器：

axios.interceptors.request.use(

    function(config){

        //在发送请求之前做什么;

        return config;

    },

    function(config){

        //对请求错误做什么;

        return Promise.reject(error);

    }

)

3.添加响应拦截器：

axios.interceptors.response.use(

    function (response) {


        //对响应数据做点什么;

        return response;

}, 

    function (error) {

        //对响应错误做点什么;

        return Promise.reject(error);
    }

)

4.config表示axios中的配置对象;

5.拦截器只对当前的实例有效;
