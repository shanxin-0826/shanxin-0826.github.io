Math

Math​

1.Math是一个工具类;

2.Math中为我们提供了数学运算相关的一些常量和方法;

常量​

Math.PI; 圆周率

方法​

Math.abs(); 求一个数的绝对值;

Math.min(); 求多个值中的最小值;

Math.max(); 求多个值中的最大值;

Math.pow(); 求x的y次幂，等价于 x ** y;

Math.sqrt(); 求一个数的平方根，等价于 x ** 0.5;

Math.floor(); 向下取整;

Math.ceil(); 向上取整;

Math.round(); 四舍五入取整;

Math.trunc(); 直接去除小数位;

Math.random(); 生成一个0-1之间的随机数;

生成 0-x 之间的随机数：

Math.round( Math.random() * x );

Math.floor( Math.random() * ( x + 1) );

生成 x-y 之间的随机数：

Math.round( Math.random() * ( y - x) + x );
