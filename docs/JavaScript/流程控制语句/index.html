流程控制语句

代码块​

1.使用{}来创建代码块，代码块可以用来对代码进行分组;

2.同一个代码块中的代码，就是同一组代码;

一个代码块中的代码要么都执行要么都不执行;

let 和 var​

1.在JS中，使用let声明的变量具有块级作用域;

在代码块中声明的变量无法再代码块的外部访问;

2.使用var声明的变量，不具有块级作用域;

流程控制语句​

流程控制语句可以用来改变程序执行的顺序;

1.条件判断语句​

if语句 语法：

    if(条件表达式){

        语句...   

    }

执行流程：

(1).if语句执行会先对if后的条件表达式进行求值判断;

如果结果为true，则执行if后的语句，如果为false则不执行;

(2).if语句只会控制紧随其后的那一行代码，如果希望可以控制多行;

可以使用{}将语句括起来;

即使if后只有1行代码，我们也应该编写代码块，这样结构会更清晰;

(3).如果if后的条件表达式不是布尔值，会转换为布尔值然后再运算;

2.条件分支语句​

if-else语句 语法：

    if(条件表达式){

        语句...

    }else{

        语句...

    }

执行流程：

if-else执行时，先对条件表达式进行求值判断;

如果结果为true，则执行if后的语句;

如果结果为false。则执行else后的语句;

if-else if-else语句 语法：

    if(条件表达式){

        语句...   

    }else if(条件表达式){

        语句...

    }else{

        语句...

    }

执行流程：

if-else if-else语句，会自上向下依次对if后的条件表达式进行求值;

如果条件表达式结果为true，则执行当前if后的语句，执行完毕结束;

如果条件表达式结果为false。则继续向下判断，直到找到true为止;

如果所有的条件表达式都是false，则执行else后的语句;

注意：

if-else if-else语句中只会有一个代码块被执行;

一旦有执行的代码块，下边的条件都不会再继续判断了;

所以一定要注意条件的编写顺序;

switch语句 语法：

    switch（表达式）{

        case 表达式：

            代码...

            break；
    
        case 表达式：

            代码...

            break；

        default：

            代码...

            break；

    }

执行流程：

(1).switch语句在执行时;

会依次将switch后的表达式和case后的表达式进行全等比较;

(2).如果比较结果为true，则自当前case处开始执行代码;

(3).如果比较结果为false，则继续比较其他case后的表达式;

直到找到true为止;

(4).如果所有的比较都是false，则执行default后的语句;

注意：

(1).当比较结果为true时，会从当前case处开始执行代码;

也就是说case是代码执行的起始位置;

(2).这就意味着只要是当前case后的代码，都会执行;

可以使用break来避免执行其他的case;

总结：

switch语句和if语句的功能是重复，switch能做的事if也能做;

反之亦然，它们最大的不同在于，switch在多个全等判断时;

结构比较清晰;

3.循环语句​

通过循环语句可以使指定的代码反复执行;

JS中一共有三种循环语句;

    while语句;

    do-while语句;

    for语句;

while循环 语法：

    while(条件表达式){
    
        语句...

    }

执行流程：

(1).while语句在执行时，会先对条件表达式进行判断;

(2).如果结果为true，则执行循环体，执行完毕，继续判断;

(3).如果为true，则再次执行循环体，执行完毕，继续判断，重复;

(4).直到条件表达式结果为false时，循环结束;

注意：

当一个循环条件表达式恒为true时，这个循环就是第一个死循环;

do-while循环 语法：

    do{
    
        语句...

    }while(条件表达式)

执行流程：

(1).do-while语句在执行时，会先执行do后的循环体;

执行完毕后，会对while后的条件表达式进行判断;

(2).如果为false，则循环终止;

(3).如果为true，则继续执行循环体，以此类推;

和while的区别：

(1).while语句是先判断再执行，do-while语句是先执行再判断;

(2).do-while语句可以确保循环至少执行一次;

for循环 语法：

    for(1.初始化表达式; 2.条件表达式; 4.更新表达式){
    
        3.语句...

    }

for循环和while没有本质区别，都是用来反复执行代码;

不同点就是语法结构，for循环更加清晰;

执行流程：

(1).执行初始化表达式，初始化变量;

(2).执行条件表达式，判断循环是否执行(true执行，false终止);

(3).判断结果为true，则执行循环体;

(4).执行更新表达式，对初始化变量进行修改;

(5).重复(2)，直到判断为false为止;

其它：

(1).初始化表达式，在循环的整个声明周期中只会执行1次;

(2).for循环中的三个表达式都可以省略;

(3).使用let在for循环的()中声明的变量是局部变量;

只能在for循环内部访问;

使用var在for循环中的()中声明的变量可以在for循环的外部访问;

(4).创建死循环的方式：

    while(1){}

    for(;;){}

break和continue​

break

1.break用来终止switch和循环语句;

2.break执行后，当前的switch或循环会立刻停止;

3.break会终止离他最近的循环;

continue

1.continue用来跳过当次循环;
