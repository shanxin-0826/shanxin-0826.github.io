面向对象

面向对象编程（OOP）​

1.程序是干嘛的？

程序就是对现实世界的抽象(照片就是对人的抽象);

2.对象是干嘛的？

一个事物抽象到程序中后就变成了对象;

在程序的世界中，一切皆对象;

3.面向对象编程？

面向对象编程是指，程序中的所有操作都是通过对象来完成;

做任何事情之前都需要先找到它的对象;

然后通过对象来完成各种操作;

面向对象的特点：​

封装、继承、多态;

封装 -- 安全性

继承 -- 扩展性

多态 -- 灵活性

封装​

1.对象就是一个用来存储不同属性的容器;

2.对象不仅存储属性，还要负责数据的安全;

3.直接添加到对象中的属性并不安全，因为它们可以被任意的修改;

4.如何确保数据的安全：

(1).私有化数据

将需要保护的数据设置为私有，只能在类内部使用;

(2).提供setter和getter方法来开放对数据的操作;

属性设置私有，通过setter和getter方法操作属性带来的好处:

①.可以控制属性的读写权限;

②.可以在方法中对属性的值进行验证;

封装总结​

1.封装主要用来保证数据的安全;

2.实现封装的方式：

(1).属性私有化 加#;

注意：私有化数据必须先声明，然后才能在内部访问;

(2).通过getter和setter方法来操作属性;

    get 属性名(){

        return this.#属性;
    
    }

    set 属性名(参数){

        this.#属性名 = 参数;

    }

多态​

1.在JS中不会检查参数的类型;

所以这就意味着任何数据都可以作为参数传递;

2.要调用某个函数，无需指定的类型，只要对象满足某些条件即可;

3.如果一个东西走路像鸭子，叫起来像鸭子，那么它就是鸭子;

4.多态为我们提供了灵活性;

继承​

1.可以通过extends关键字来完成继承;

2.当一个类继承另一个类时;

就相当于将另一个类中的代码复制到了当前类中（简单理解）;

3.继承发生时;

被继承的类称为父类（超类），继承的类称为子类;

4.通过继承可以减少重复的代码;

并且可以在不修改一个类的前提下对其进行扩展;

5.OCP 开闭原则

程序应该对修改关闭，对扩展开放;

6.在子类中，可以通过创建同名方法来重写父类的方法;

7.重写构造函数时，构造函数的第一行代码必须为super();

super()可以帮我们调用父类的构造函数;

8.在方法中可以使用super来引用父类的方法;

super.函数名();

总结​

面相对象的本质就是，编写代码时所有的操作都是通过对象进行的;

面相对象的编程步骤：

    1.找对象;

    2.搞对象;

学习对象：

    1.明确这个对象代表什么，有什么用;

    2.如何获取到这个对象;

    3.如何使用这个对象(对象中的属性和方法);

对象的分类：

    内建对象：

    由ES标准所定义的对象;

    比如 Object Function String Number...

    宿主对象：

    由浏览器提供的对象;

    BOM、DOM;

    自定义对象：

    由开发人员自己创建的对象;
