数据类型

数值（Number）​

1.在JS中所有的整数和浮点数都是Number类型;

2.JS中的数值并不是无限大的，当数值超过一定范围后会显示近似值;

3.Infinity 是一个特殊的数值表示无穷;

4.所以在JS中进行一些精度比较高的运算时要十分注意;

5.NaN 也是一个特殊的数值，表示非法的数字;

大整数（BigInt）​

1.大整数用来表示一些比较大的整数;

2.大整数使用n结尾，它可以表示的数字范围是无限大;

其他进制的数字：

二进制 0b

八进制 0o

十六进制 0x

字符串（String）​

1.在JS中使用单引号或双引号来表示字符串;

2.转译字符 \

\" --> "

\' --> '

\\ --> \

\t --> 制表符

\n --> 换行

模板字符串

1.使用返单引号 ` 俩表示模板字符串;

2.模板字符串中可以嵌入变量;

布尔值（Boolean）​

1.布尔值主要用来进行逻辑判断;

2.布尔值只有两个 true 和 false;

3.使用typeof检查一个布尔值时会返回 "boolean";

空值（Null）​

1.空值用来表示空对象;

2.空值只有一个 null;

3.使用typeof检查一个空值时会返回 "object";

4.使用typeof无法检查空值;

未定义（Undefined）​

1.当声明一个变量而没有赋值时，它的值就是undefined;

2.undefined类型的值只有一个就是undefined;

3.使用typeof检查一个undefined类型的值时会返回 "undefined";

符号（Symbol）​

1.用来创建一个唯一的标识;

2.使用typeof检查符号时会返回 "symbol";

JS中原始值一共有七种​

1.Number

2.BigInt

3.String

4.Boolean

5.Null

6.Undefined

7.Symbol

七种原始值是构成各种数据的基石;

原始值在JS中是不可变类型，一旦创建就不能修改;

强制类型转换​

转为字符串​

1.调用toString()方法将其他类型转换为字符串;

由于null和undefined中没有toString();

所以对这两个类型调用toString()时会报错;

2.调用String()函数将其他类型转换为字符串;

对于拥有toString()方法的值调用String()函数时;

实际上就是在调用toString()方法;

对于null，则直接转换为"null";

对于undefined，则直接转换为"undefined";

转为数值​

1.使用Number()函数来将其他类型转换为数值;

2.转换的情况：

(1).字符串：

如果字符串是一个合法的数字，则会自动转换为对应的数值;

如果字符串不是合法的数字，则转换为NaN;

如果字符串是空串或纯空格的字符串，则转换为0;

(2).布尔值：

true转换为1，false转换为0;

(3).null 转换为 0;

(4).undefined 转换为 NaN;

3.专门用来将字符串转换为数值的两个方法：

parseInt（）将一个字符串转换为一个整数;

1.解析时，会自左向右读取一个字符串;

直到读取到字符串中所有的有效整数;

2.也可以使用parseInt()来对一个数字进行取整;

parseFloat（）将一个字符串转换为浮点数;

1.解析时，会自左向右读取一个字符串;

直到读取到字符串中所有的有效小数;

转为布尔值​

1.使用Boolean()函数来将其他类型转换为布尔值;

2.转换的情况：

(1).数字：

0 和 NaN 转换为false;

其余是true;

(2).字符串：

空串 转换为 false;

其余是true;

(3).null和undefined 都转换为 false;

(4).对象转换为true;

3.所有表示空性的没有的错误的值都会转换为false：

0 NaN 空串 null undefined false;
